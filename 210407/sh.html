jquery 적용하기

                    jquery를 이용하기 위해서는 jquery가 구현되어있는 js파일을 페이지에 불러와서 사용함
  • CDN : http 통신으로 제공하는 서버에서 페이지 로드 시 받아오는 방식
  • 파일 다운로드 : 직접 다운로드해서 프로젝트에 넣고 불러오는 방식

Element불러오기

                    (id, 전체, )
