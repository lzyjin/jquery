:first, :last, even, odd

선택자:first : 선택자로 선택된 element 중에서 0번 인덱스의 요소를 불러옴
선택자:last : 선택자로 선택된 element 중에서 마지막 인덱스의 요소를 불러옴
선택자:even : 선택자로 선택된 element 중에서 짝수 인덱스의 요소를 불러옴
선택자:odd : 선택자로 선택된 element 중에서 홀수 인덱스의 요소를 불러옴

:eq, :gt, :lt

  • 선택자:eq(n) : 선택된 태그 중 n번째 인덱스의 태그를 불러옴
  • 선택자:gt(n) : 선택된 태그 중 n번째 인덱스보다 큰 인덱스를 가진 태그들을 불러옴
  • 선택자:lt(n) : 선택된 태그 중 n번째 인덱스보다 작은 인덱스를 가진 태그들을 불러옴
